<h1>Im-not-a-robot-level-controller</h1><br>
Adds buttons to choose a level, go to last level, skip the final level video, and go to next level/previous level.<br>

Get Tampermonkey.<br>
Links for each browser:<br>

<a href="https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo">Google Chrome</a><br>,
<a href="https://addons.mozilla.org/en-US/firefox/addon/tampermonkey">Mozilla Firefox</a><br>,
<a href="https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd">Microsoft Edge</a><br>,
or
<a href="https://addons.opera.com/extensions/details/tampermonkey-beta">Opera GX</a><br>.

Copy this script, and in Tampermonkey create a new script, and paste this script. Click Ctrl+S or smth else to save, and go to <a href="https://neal.fun/not-a-robot">I'm Not a Robot!</a><br>
